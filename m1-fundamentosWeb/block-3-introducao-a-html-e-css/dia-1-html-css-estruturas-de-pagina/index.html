Estudante de desenvolvimento Web FullStack

Trabalhei em outras áreas por muito tempo e busco a transição de carreira devido não somente por ser uma carreira promissora mas também por que a profissão me atrai. Cursei Web Designer em 2004 porque precisava aprender Corel Draw e Photoshop. Nem dei muito valor na época às páginas que eram estáticas com o HTML puro. Era divertido. Em 2014 comecei a ter contato com linguagem de programação de mais baixo nível através da VBA do excel que utiliza linguagem semelhante a C#. Depois que aprendi a lógica de programação não parei mais de me interessar ... Venho aprendendo novas ferramentas, linguagens e frameworks a cada novo dia.
